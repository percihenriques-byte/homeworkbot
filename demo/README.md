# Demo do Homework Assistant (rodar no seu PC, sem login)

Quer só **ver o app funcionando** no seu computador, sem cadastro nenhum?

## Como usar (Windows)

1. Abra esta pasta (`demo`).
2. Dê **dois cliques** em **`INICIAR-DEMO.bat`**.
3. Espere. **Na primeira vez** ele baixa sozinho o que precisa (Node + banco de dados) —
   pode levar **alguns minutos** e ~120 MB. Nas próximas vezes abre rapidinho.
4. O app abre **sozinho no seu navegador**, já logado como "Visitante", com tarefas de exemplo.

Para **desligar**: feche a janela preta que abriu.

## O que dá pra ver

- Painel, Minhas Tarefas, a aba **"Feitas pela IA"**, Cronograma, Ferramentas, Memórias.
- O **Jarvis (IA)** responde (com uma IA de demonstração — quando você configurar a chave grátis
  do Google Gemini no app publicado, as respostas viram reais).

## Importante

- É só uma **demonstração local**: roda só neste computador e só enquanto a janela estiver aberta.
  Não dá pra abrir do celular do seu filho.
- Para o seu filho usar **de qualquer lugar** (celular, escola), é preciso **publicar online** —
  veja o guia `COMO-PUBLICAR-DE-GRACA.md` na pasta principal (deploy grátis em 1 botão).
- Tudo que ele baixa fica em `demo/.runtime` (pode apagar essa pasta pra "resetar" o demo).

## Se o Windows reclamar

Se aparecer um aviso de segurança do Windows/antivírus ao rodar o `.bat`, é porque ele é um
script novo (não é vírus — é este projeto). Escolha "Mais informações" → "Executar assim mesmo".
